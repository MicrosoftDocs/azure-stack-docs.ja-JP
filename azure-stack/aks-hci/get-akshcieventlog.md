# <a name="get-akshcieventlog"></a>Get-AksHciEventLog

## <a name="synopsis"></a>構文
Azure Kubernetes Service on Azure Stack HCI PowerShell モジュールからすべてのイベント ログを取得します。

## <a name="syntax"></a>構文

```powershell
Get-AksHciEventLog
```

## <a name="description"></a>説明
Azure Kubernetes Service on Azure Stack HCI PowerShell モジュールからすべてのイベント ログを取得します。

## <a name="examples"></a>例

### <a name="example"></a>例
```powershell
PS C:\> Get-AksHciEventLog
```
