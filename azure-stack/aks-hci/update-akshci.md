# <a name="update-akshci"></a>Update-AksHci

## <a name="synopsis"></a>構文
Azure Kubernetes Service ホストを最新の Kubernetes バージョンに更新します。

## <a name="syntax"></a>構文

```powershell
Update-AksHci
```

## <a name="description"></a>説明
Azure Kubernetes Service ホストを最新の Kubernetes バージョンに更新します。

## <a name="examples"></a>例

### <a name="example"></a>例
```powershell
PS C:\> Update-AksHci
```
