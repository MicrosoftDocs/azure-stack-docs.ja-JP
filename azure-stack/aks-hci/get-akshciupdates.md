# <a name="get-akshciupdates"></a>Get-AksHciUpdates

## <a name="synopsis"></a>構文
Azure Kubernetes Service on Azure Stack HCI 用の利用可能な Kubernetes 更新プログラムを一覧表示します。

## <a name="syntax"></a>構文

```powershell
Get-AksHciUpdates
```

## <a name="description"></a>説明
Azure Kubernetes Service on Azure Stack HCI 用の利用可能な Kubernetes 更新プログラムを一覧表示します。

## <a name="examples"></a>例

### <a name="example"></a>例
```powershell
PS C:\> Get-AksHciUpdates
```
