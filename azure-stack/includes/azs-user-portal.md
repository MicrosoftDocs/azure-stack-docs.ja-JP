ユーザー ポータルにサインインします。
    - 統合システム デプロイの場合、ポータル アドレスはソリューションのリージョンと外部ドメイン名によって変わります。 形式は、 https://portal.&lt;*リージョン*&gt;.&lt;*FQDN*&gt; です。
